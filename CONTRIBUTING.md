# การร่วมเพิ่มคำ

กรุณาส่งคำผ่าน [แบบฟอร์มเสนอคำ](https://github.com/Yobby1997/EnonE-Thai-Thesaurus/issues/new?template=suggest-word.yml) แทนการส่งไฟล์ฐานข้อมูลทั้งก้อนหรือแก้ `data/thesaurus.json` โดยตรง

ขั้นตอนของคลังคำคือ **เสนอ → ตรวจคำซ้ำและบริบท → รีวิว → รวมเข้าคลังกลาง → ทดสอบ → เผยแพร่** วิธีนี้ทำให้สำเนาเก่าจากเครื่องใดเครื่องหนึ่งไม่สามารถทับข้อมูลล่าสุดได้

ผู้ดูแลเท่านั้นที่นำข้อเสนอซึ่งผ่านการตรวจแล้วไปแก้ข้อมูลต้นทาง จากนั้นต้องรัน:

```powershell
npm test
npm run build
```

รายละเอียดการเรียก API และหลักเกณฑ์ของคำใกล้เคียงอยู่ที่ `/docs` บนเซิร์ฟเวอร์ของโครงการ
